🚀 TaskFlow

### Organize Today. Achieve Tomorrow.

📖 About
TaskFlow is a Django-based task management web application designed to help users organize, prioritize, and manage their daily tasks efficiently.

✨ Features
- User Authentication
- Create, Update & Delete Tasks
- Mark Tasks as Complete
- Search Tasks
- Filter Tasks
- Sort Tasks
- Due Date Tracking
- Task Status Indicators
- Dashboard Statistics
- Completion Percentage
- Responsive UI

🛠 Tech Stack
- Python
- Django
- Bootstrap 5
- HTML
- CSS
- SQLite
